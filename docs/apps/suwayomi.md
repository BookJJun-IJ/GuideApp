# Suwayomi — Manga Reader

Suwayomi is a self-hosted manga reader that connects to over 1,200 manga sources. Read manga in your browser without ads or tracking.

## Installing Suwayomi

1. Open the **App Store**
2. Search for **Suwayomi**
3. Click **Install**
4. Wait for the installation to complete

## First-Time Setup

1. Open Suwayomi from your dashboard
2. The web interface is ready to use immediately — no account setup needed

## Adding Manga Sources (Extensions)

Before you can browse manga, you need to enable sources:

1. Go to **Browse** in the sidebar
2. Click the **Extensions** tab
3. Browse the available extensions by language
4. Click **Install** next to the sources you want

Popular sources include:

| Source | Language | Description |
|--------|----------|-------------|
| MangaDex | Multi | Largest free manga library |
| MangaSee | English | Popular English manga |
| Webtoons | Multi | Korean webtoons |
| Manga Plus | Multi | Official Shueisha titles |

## Browsing and Reading

1. Go to **Browse** in the sidebar
2. Select an installed source
3. Browse or search for manga
4. Click a title to see details
5. Click a chapter to start reading

## Adding to Library

1. Open a manga's detail page
2. Click **Add to Library**
3. The manga will appear in your **Library** tab
4. New chapters will be tracked automatically

## Downloading for Offline Reading

1. Open a manga from your library
2. Select chapters you want to download
3. Click the **Download** button
4. Downloaded chapters are stored in `/DATA/Media/Mangas/`

> **Note:** Downloaded manga can also be accessed through the File Manager.

## Reading Settings

While reading, you can customize the experience:

- **Reading Mode**: Left-to-right, right-to-left, or vertical scrolling
- **Page Layout**: Single page, double page, or continuous
- **Image Scaling**: Fit width, fit height, or original size

Access these settings by clicking the gear icon while reading a chapter.

## Tracking Progress

Suwayomi automatically tracks which chapters you've read. Your reading position is saved so you can pick up where you left off.

## Cloudflare-Protected Sources

Some manga sources use Cloudflare protection. Your PCS includes **Byparr** (a Cloudflare bypass tool) that handles this automatically. If a source isn't loading:

1. Wait a few seconds — Byparr will solve the challenge
2. Try refreshing the page
3. If it persists, check that the Byparr service is running in your dashboard
