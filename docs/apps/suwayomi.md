# Suwayomi — Read Manga Without Ads

Suwayomi connects to 1,200+ manga sources. Read in your browser, no ads, no tracking.

## Getting Started

1. Open **App Store** and install **Suwayomi**
2. Open Suwayomi — no account needed, it's ready to go

## Adding Sources

Before you can read, you need to add manga sources:

1. Go to **Browse > Extensions**
2. Pick sources by language
3. Click **Install** on the ones you want

Some popular picks:

| Source | What it has |
|--------|------------|
| **MangaDex** | Largest free library |
| **Manga Plus** | Official Shueisha titles |
| **Webtoons** | Korean webtoons |

## Reading

1. Go to **Browse** and select a source
2. Search for a manga
3. Click a chapter to start reading

## Saving to Library

Click **Add to Library** on any manga's page. New chapters are tracked automatically.

## If a Source Won't Load

Some sources use Cloudflare protection. Your PCS has **Byparr** which handles this automatically. Give it a few seconds — if it still doesn't load, try refreshing.
