# Docmost — Your Own Notion

Docmost is a private wiki and note-taking app. Write notes, build documentation, organize everything.

## Getting Started

1. Open **App Store** and install **Docmost**
2. Open Docmost and create your account
3. Start writing

## Creating Pages

1. Click **+ New Page** in the sidebar
2. Start typing — changes save automatically
3. Type `/` to see all block types (headings, lists, code, tables, etc.)

## Organizing

Drag pages in the sidebar to nest them:

```
My Workspace
  Personal Notes
    Daily Journal
    Reading List
  Work
    Projects
    Meeting Notes
```

Use **Spaces** to separate big topics (Personal, Work, Recipes).

## Search

Press `Ctrl+K` to search across all your pages.

## Sharing

1. Open a page
2. Click **Share** in the top-right
3. Turn on **Public Access**
4. Copy the link — anyone can read it, no account needed

## Inviting Others

Go to **Settings > Members > Invite Member**. You can set them as Admin, Editor, or Viewer.
